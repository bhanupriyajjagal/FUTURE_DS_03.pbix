# 📊 Marketing Funnel & Conversion Performance Analysis

## 📌 Project Overview

This project analyzes marketing funnel performance using **Power BI** to understand how users move through the customer journey—from being contacted to becoming customers. The dashboard helps identify conversion rates, drop-off points, and high-performing customer segments to support data-driven marketing decisions.

---

## 🎯 Objectives

- Analyze marketing campaign performance
- Measure visitor-to-customer conversion
- Identify funnel drop-off points
- Compare customer conversions across different segments
- Provide actionable recommendations to improve campaign effectiveness

---

## 🛠️ Tools Used

- **Power BI**
- Power Query (Data Cleaning)
- DAX (Measures & KPIs)

---

## 📂 Dataset

**Dataset:** Bank Marketing Dataset

The dataset contains customer information such as:

- Age
- Job
- Marital Status
- Education
- Contact Type
- Month
- Housing Loan
- Personal Loan
- Campaign Details
- Previous Marketing Outcome
- Customer Subscription (Target Variable)

- Identified the stages where customer drop-offs occur.
- Compared conversion performance across customer segments.
- Determined which marketing channels and customer profiles generate better conversions.
- Highlighted opportunities to improve future marketing campaigns.

---

## 💡 Recommendations

- Focus marketing efforts on high-converting customer groups.
- Improve campaigns during low-conversion periods.
- Optimize communication channels with better response rates.
- Personalize campaigns based on customer demographics.
- Reduce funnel drop-offs through targeted engagement strategies.
